# 참고 링크 및 코드 개선

re.sub함수에 대해 검색해보니 '[^\w\s]' 부분은 모든 알파벳과 숫자, 그리고 공백을 제외한 문자열을 뜻하지만 언더바(_)도 제외되는 문자열에 포함한다는 것을 발견했습니다. 혹시나 모를 에러발생을 막기위해 언더바도 대체하기를 원한다면
```clean = re.sub('[^a-zA-Z0-9\s]', '', s)```
로 수정하는것도 좋을 것같습니다.
